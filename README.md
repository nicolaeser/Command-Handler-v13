# Use 


## Start Bot 
▶ Running Deploy Commands & Index 
```
npm run bot
``` 



▶ Only Running Index
```
npm run index
``` 



▶ Only Running Deploy Commands
```
npm run deployCommands
``` 


## Package Installation
```
npm i fs discord.js @discordjs/builders @discordjs/rest discord-api-types
```
